# installing [uGet]
Use `sudo add-apt-repository ppa:plushuang-tw/uget-stable -y && sudo apt update && sudo apt install uget -y` to install from [Ubuntu PPA](https://en.wikipedia.org/wiki/Ubuntu_(operating_system)#Package_Archives).
