# installing Git
To install from [Ubuntu PPA](https://en.wikipedia.org/wiki/Ubuntu_(operating_system)#Package_Archives), use `sudo add-apt-repository ppa:git-core/ppa -y && sudo apt-get update ; sudo apt-get install git -y`.
